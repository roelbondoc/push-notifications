# push-notifications

## requirements

* [docker](https://docs.docker.com/install/)
* [docker-compose](https://docs.docker.com/compose/install/)

## setup

1. Clone repo
2. Build project
  ```
  $ docker-compose run app vapor build
  ```
3. Run services
  ```
  $ docker-compose up --build
  ```
4. Open browser to [localhost:8080](http://localhost:8080)
